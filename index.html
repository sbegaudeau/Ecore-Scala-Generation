Model driven Scala development

Need reasons to love this generator? Look no further.

Open Source

Not only this generator is completely open source and as such, you can download its source code or fork it but it also contains a developer journal with the details of the conception of this generator giving all the informations to start your own code generator.

Integrated in Eclipse

This generator is seamlessly integrated in Eclipse with a full fledged user interface. From the internal documentation to the run configuration, you can deliver this generator to your end users without having to train them to code generation.

Customizable

What good is a code generator if you can't influence the generated code? With this Scala generator, you can parameterized the generation to generate Scala code following your conventions.
